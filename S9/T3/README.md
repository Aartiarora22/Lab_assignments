## Problem Statement
Given an array of numbers of input. Find the most frequently occurring number 
in the array using HashMap. If more than 1 number has the highest frequency, 
print all the numbers.



**Sample Test Case**
```
Input:
{10,20,30,20,40,50}

Output:
20
```
