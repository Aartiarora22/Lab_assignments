## Problem Statement
Given a string as input. Print capital if we encounter capital letter. Print small 
otherwise. Assume string contains only alphabets.

**Sample Test Case**
```
Input:
"car"

Output:
c:small
a:small
r:small
```
## Other Test cases

```
Input:
"TriSecT"

Output:

S:capital
e:small
c:small
T:capital

```
