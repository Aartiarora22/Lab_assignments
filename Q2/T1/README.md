## Problem Statement
Given a string as input. Give count of character 'x' and 'y' in given string

**Sample Test Case**
```
Input:
"exactly"

Output:
x#1
y#1
```
## Other Test cases
**Test Case 1**
```
Input:
"myday"

Output:
x#0
y#2
```
