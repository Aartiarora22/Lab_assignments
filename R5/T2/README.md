## Problem Statement
Given two input arrays as input. Create a new array containing all elements from 
both the input arrays except multiples of 2 and 3.
Print this new array along with its size.

**Sample Test Case**
```
Input:
{1,2}
{3,4,5} 

Output:
New array size:2
1
5

```
