## Problem Statement
Create a function named power that takes two integers n and m as input and returns 
the square of variable n when m is odd and returns cube of n when m is even.


**Sample Test Case**
```
Input:
n=2 ,m=12

Output:
8
```

Output:

```

Output:

```
