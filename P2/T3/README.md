## Problem Statement
Given two integers as input. Print the sum of 4th power of all numbers from 1 to n.

**Sample Test Case**
```
Input:
2

Output:
17

Explanation:
n = 2
4th power of 1 = 1 X 1 X 1 X 1 = 1
4th power of 2 = 2 X 2 X 2 X 2 = 16
sum = 1+16 = 17
```
