# Product of all numbers from 1 to n

## Problem Statement
You're given n as input. Print product of all numbers from 1 to n.

**Sample Test Case**
```
Input:
5

Output:
120

Explanation:
n = 5
product = 1*2*3*4*5 = 120
```
