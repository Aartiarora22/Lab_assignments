## Problem Statement
Create a function named grades that takes marks of a student in one subject as input 
and prints:
 "Grade A" if marks are at least 90
 "Grade B" if marks are at least 75
 "Grade C" if marks secured are at least 40
 "Fail" otherwise



**Sample Test Case**
```
Input:
94

Output:
Grade A
```
