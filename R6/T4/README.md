## Problem Statement
Given an array of numbers as input. Print numbers that are divisible by 3 and 4 
but not by 12. Also these numbers should lie in the range of 3 and 99 [inclusive].

**Sample Test Case**
```
Input:
{9,44,100,104}

Output:
9
44
```
## Other Test cases

Output:

```
**Test Case 2**
```

{3,12,24}
