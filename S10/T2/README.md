## Problem Statement
Given an array of numbers as input, use HashSet to print all the unique numbers in 
the array. The output can be printed in any order


**Sample Test Case**
```
Input:
{1,3,5,7,9,4,7,11,9}

Output:
1,3,5,7,9,4,11
```
## Other Test cases
**Test Case 1**
```
Input:
{10,20,30,40,50}

10,20,30,40,50
```
**Test Case 2**
```
Input:
{9,9,9,9,1,2,3,9}

Output:
9,1,2,3
```
