## Problem Statement
Given an array of numbers as input. Create a HashSet of all the duplicate 
numbers in the array. Also create an ArrayList of all the duplicate numbers from 
the HashSet. Print the ArrayList.


**Sample Test Case**
```
Input:
{1,2,3,2,4,5,1,6,1}

Output:
1,2
```
