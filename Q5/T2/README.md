## Problem Statement
Given an array of numbers as input where each element is less than 1000. Print 
the number which is nearest to 1000

**Sample Test Case**
```
Input:
{179,256,9}

Output:
256
```
## Other Test cases
**Test Case 1**
```
Input:
{919,999,979,899}

Output:
999
```
**Test Case 2**
```
Input:
{0,8,1,5,3}

Output:
8
```
