## Problem Statement
Given a string as input. Create a new string in which every occurrence of digit is 
replaced by '#' and every capital letter is doubled.

**Sample Test Case**
```
Input:
"Top20"

Output:
"TTop##"
```
## Other Test cases
**Test Case 1**
```
Input:
"car"

Output:
"car"
```
**Test Case 2**
```
Input:
"HellO"

Output:
"HHellOO"
```
**Test Case 3**
```
Input:
"1GOAL9"

Output:
"#GGOOAALL#"
```
