# Sum of Cube of all numbers from 1 to n.

Given n as input. Write a code to print sum of cube of all numbers from 1 to n.

**Sample Test Case**
```
Input:
2

Output:
9

Explanation:
Cube of 1- 1
Cube of 2- 8
Sum of 2 numbers ->1 + 8 = 9
```
