## Problem Statement
Create a function named encode that takes an integer n and a string as input. If:
 n = 0 then it replaces every character at even index with #
 n = 1 then it replaces every character at odd index with #


**Sample Test Case**
```
Input:
hello", 0

Output:
#e#l#
```
## Other Test cases
**Test Case 1**
```
Input:
"trisect",1

Output:
t#i#e#t
```
