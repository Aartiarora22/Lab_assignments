## Problem Statement
Create a function named countLowerCase that takes a String as input and prints the 
total number of lowercase characters in input string.


**Sample Test Case**
```
Input:
"Trisect"

Output:
Lowercase=6
```

```
Input:
"HELLO"

Output:

```
**Test Case 2**
